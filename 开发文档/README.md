  - ASR引擎实现（Whisper / faster-whisper 为当前实现）
    - Faster Whisper引擎（当前实现）

  - 场景检测引擎（PySceneDetect 为当前实现）
    - TransNetV2实现（可选扩展）
    - PySceneDetect实现（当前实现）
